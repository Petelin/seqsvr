# seqsvr

## 全局唯一序列号生成服务

- 基于微信亿万级序列号生成器理论
- 为每一个entityID生成单调递增的seqID
- 千万qps(理论上可以无限扩展)
- 节点故障自动切流量
- 支持手动一键切流量，动态增加减少机器
- 可选客户端/服务端路由表，整个架构可以放在微服务下面


## 性能，单机
